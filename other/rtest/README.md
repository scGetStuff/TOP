# setup

```sh
# run in parent directory
npm create vite@latest rtest -- --template react
cd rtest
npm install
npm run dev

# clean out template CSS and jsx

# testing tool stuff
npm i -D vitest
npm i -D jsdom
npm i -D @testing-library/react @testing-library/jest-dom

```
